# Código de Conducta

Para mantener una comunidad sana y agradable, por favor cumple con estas reglas básicas:

- Sé respetuoso y profesional.
- No toleramos lenguaje ofensivo ni discriminatorio.
- Respeta la privacidad y no publiques información sensible.
- Si detectas un comportamiento inapropiado, repórtalo a los mantenedores.

Este proyecto sigue el [Contributor Covenant](https://www.contributor-covenant.org/) en su versión más reciente.
